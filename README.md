# 📊 Online Sales Data Dashboard

This project presents an Online Sales Data Dashboard built using business intelligence tools (Power BI / Excel). The dashboard provides insights into sales performance across countries, categories, payment methods, shipping providers, and time.

**Project Title**: Online Sales Data Dashboard Analysis

**Level**: Beginner  

## Objectives

It enables stakeholders to:

Monitor total sales performance

Analyze category-wise sales distribution

Track sales trends over time

Compare shipment providers and payment methods

Evaluate product-level performance

## 🗂 Dataset Description

| Column Name      | Description                                              |
| ---------------- | -------------------------------------------------------- |
| InvoiceNo        | Unique invoice number                                    |
| StockCode        | Product code                                             |
| Description      | Product name                                             |
| Quantity         | Number of items sold                                     |
| InvoiceDate      | Date of transaction                                      |
| UnitPrice        | Price per unit                                           |
| CustomerID       | Unique customer identifier                               |
| Country          | Country of purchase                                      |
| Discount         | Discount applied                                         |
| PaymentMethod    | Mode of payment (Bank Transfer, PayPal, Credit Card)     |
| ShippingCost     | Shipping charges                                         |
| Category         | Product category (Electronics, Apparel, Furniture, etc.) |
| SalesChannel     | Online / In-store                                        |
| ReturnStatus     | Returned / Not Returned                                  |
| ShipmentProvider | DHL, FedEx, UPS, Royal Mail                              |
| Warehouse        | Shipping warehouse                                       |
| OrderPriority    | Low, Medium, High                                        |
